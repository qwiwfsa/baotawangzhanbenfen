# 资金业务官网 - 设计系统规范

## 1. 设计理念

### 整体调性
- **风格定位**: 高端金融杂志感，专业权威但不失温度
- **视觉语言**: 大量留白、克制动效、温暖中性色调
- **核心关键词**: 信任、专业、稳健、高端

### 设计策略
- 采用 Editorial Luxury 风格：温暖米白底色 + 深褐点缀
- 杂志感排版：大标题、宽松字间距、非对称布局
- Bento Grid 业务展示：模块化、信息层次分明

## 3. 字体系统

### 字体家族
```css
/* 中文标题 */
--font-display: "Noto Serif SC", "Source Han Serif SC", "STSong", serif;

/* 中文正文 */
--font-body: "Noto Sans SC", "Source Han Sans SC", "PingFang SC", "Microsoft YaHei", sans-serif;

/* 英文/数字 */
--font-en: "DM Sans", "Inter", -apple-system, sans-serif;

/* 等宽字体（数据展示） */
--font-mono: "JetBrains Mono", "SF Mono", monospace;
```

## 7. 动效规范

### 缓动函数
```css
/* 标准过渡 */
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);

/* 入场动画 */
--ease-enter: cubic-bezier(0.16, 1, 0.3, 1);

/* 退场动画 */
--ease-exit: cubic-bezier(0.4, 0, 1, 1);

/* 弹性 */
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);
```

### 时长规范
| 类型 | 时长 | 用途 |
|-----|------|------|
| **微交互** | 150ms | 按钮hover、链接变色 |
| **标准** | 300ms | 卡片hover、展开收起 |
| **入场** | 600ms | 元素进入视口 |
| **页面** | 800ms | 页面切换、大模块 |

### 滚动动画
- 元素进入视口时执行淡入+上移
- 初始状态: `opacity: 0; transform: translateY(20px)`
- 结束状态: `opacity: 1; transform: translateY(0)`
- 时长: 600ms，缓动: `cubic-bezier(0.16, 1, 0.3, 1)`
- 错开延迟: 每个元素间隔 80ms

## 8. 组件规范

### 按钮

#### 主按钮（Primary）
```
背景: #3D3229
文字: #FFFFFF
内边距: 16px 32px
圆角: 4px
字体: 15px, 500字重, 字间距0.05em
Hover: 背景#5A4D40, transform: translateY(-1px)
Active: transform: scale(0.98)
```

#### 次按钮（Secondary）
```
背景: transparent
边框: 1px solid #3D3229
文字: #3D3229
内边距: 16px 32px
圆角: 4px
Hover: 背景#F5EFE6
```

#### 文字按钮（Text）
```
背景: transparent
文字: #3D3229
下划线: 1px solid #B8956A（hover时显示）
Hover: 文字颜色#B8956A
```

### 卡片

#### 标准卡片
```
背景: #FFFFFF
边框: 1px solid #E8E6E1
圆角: 16px
内边距: 32px
阴影: 0 1px 3px rgba(61, 50, 41, 0.04)
Hover: 阴影加深, transform: translateY(-2px)
```

#### 特色卡片（Bento）
```
背景: #FFFFFF 或 #F5EFE6（强调）
边框: 1px solid #E8E6E1
圆角: 16px
内边距: 40px
布局: 根据内容自适应
```

### 标签

```

文字: #3D3229
内边距: 6px 14px
圆角: 9999px（pill）
字体: 12px, 500字重, 大写, 字间距0.1em
```

### 输入框

