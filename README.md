# 恒信资本网站 - 可视化CMS系统

## 系统简介

这是一个纯前端可视化CMS内容管理系统，管理员登录后可以像编辑Word文档一样编辑网站内容。

## 功能特性

### 1. 用户认证系统
- 管理员登录/登出
- 密码保护（默认：admin / admin123）
- 会话管理

### 2. 可视化编辑器
- 点击任意元素即可编辑
- 实时预览修改效果
- 拖拽添加/删除模块
- 撤销/重做功能

### 3. 可编辑内容
- 标题、文字内容
- 字体大小、颜色、样式
- 图片替换、上传、删除
- 视频模块添加
- 卡片添加/删除/排序
- 背景颜色/图片

### 4. 模块系统
- 文本模块
- 图片模块
- 视频模块
- 卡片模块
- 按钮模块
- 分隔线模块
- 数据统计模块

### 5. 页面管理
- 首页编辑
- 子页面编辑（业务范围、成功案例等）
- 导航菜单编辑

### 6. 数据存储
- 使用 localStorage 保存修改
- 导出/导入JSON备份
- 版本历史记录

## 使用说明

### 1. 登录管理后台

访问 `admin/login.html`，使用以下默认账号登录：
- 用户名：admin
- 密码：admin123

### 2. 编辑页面内容

#### 方法一：通过管理后台
1. 登录后进入管理仪表板
2. 点击"页面管理"查看所有页面
3. 点击"编辑"按钮进入编辑模式

#### 方法二：直接访问页面
1. 在任意页面URL后添加 `?edit=true` 参数
2. 如果未登录，会提示先登录
3. 登录后自动进入编辑模式

### 3. 编辑操作

#### 选择元素
- 点击页面上的任意元素即可选中
- 选中后会显示蓝色边框和操作工具栏

#### 编辑文本
- 双击文本元素直接编辑
- 或在属性面板中修改

#### 编辑图片
- 点击图片，选择"编辑内容"
- 支持本地上传新图片

#### 修改样式
- 选中元素后，在右侧属性面板中修改：
  - 文字颜色
  - 背景颜色
  - 字体大小
  - 对齐方式
  - 内边距/外边距

#### 添加模块
- 点击顶部工具栏"添加模块"按钮
- 选择要添加的模块类型
- 新模块会添加到页面底部，可拖动调整位置

#### 删除元素
- 选中元素后，点击工具栏的删除按钮
- 或按 Delete 键

### 4. 保存更改

- 点击顶部工具栏的"保存"按钮
- 或使用快捷键 Ctrl+S
- 数据会自动保存到浏览器本地存储

### 5. 预览效果

- 点击"预览"按钮查看实际效果
- 再次点击返回编辑模式

### 6. 导出备份

- 点击"导出"按钮下载JSON备份文件
- 可用于数据迁移或恢复

### 7. 退出登录

- 点击用户菜单或退出按钮
- 退出后会清除登录状态

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl + S | 保存内容 |
| Ctrl + Z | 撤销操作 |
| Esc | 取消选择 |
| Delete | 删除选中元素 |

## 注意事项

1. **数据存储**：所有数据保存在浏览器本地存储中，清除浏览器数据会导致内容丢失，请定期导出备份。

2. **图片上传**：图片以Base64格式存储，大图片可能占用较多存储空间。

3. **多设备同步**：由于使用本地存储，不同设备或浏览器之间的数据不互通。

4. **协作编辑**：本系统为单用户设计，不支持多人同时编辑。
