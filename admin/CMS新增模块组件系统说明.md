# CMS新增模块组件系统说明

## 系统概述

本次开发为恒信资本CMS系统新增了完整的模块组件系统，包括：

1. **行业资讯卡片式文章系统**
2. **基础组件（6个）**
3. **排版组件（9个）**
4. **互动组件（8个）**
5. **文字超链接功能**
6. **拖拽式编辑器**

---

## 功能模块详解

## 使用指南

### 访问文章管理

1. 登录CMS后台
2. 点击左侧导航「文章管理」
3. 可以查看、新建、编辑、删除文章
4. 支持按分类和状态筛选

### 使用拖拽编辑器

1. 点击左侧导航「拖拽编辑器」
2. 选择要编辑的页面
3. 从左侧组件库拖拽组件到画布
4. 点击组件在右侧面板编辑属性
5. 点击「预览」查看效果
6. 点击「保存」或「发布」生效

### 在富文本中插入链接

1. 选中要添加链接的文字
2. 点击工具栏的链接按钮（或按Ctrl+K）
3. 选择「外部链接」或「内部页面」
4. 填写链接信息
5. 点击「插入链接」

## 技术架构

### 文件结构
```
admin/
├── components/
│   ├── component-config.js      # 组件配置
│   ├── component-renderer.js    # 组件渲染器
│   ├── link-dialog.js           # 超链接对话框
│   ├── basic/                   # 基础组件模板
│   ├── layout/                  # 排版组件模板
│   ├── interactive/             # 互动组件模板
│   └── news/                    # 文章系统
│       ├── article-list.html
│       ├── article-edit.html
│       └── category.html
├── builder/
│   ├── index.html               # 拖拽编辑器
│   ├── builder.css              # 编辑器样式
│   └── builder.js               # 编辑器逻辑
└── api/news/
    ├── list.php                 # 文章列表API
    ├── detail.php               # 文章详情API
    ├── save.php                 # 保存文章API
    ├── delete.php               # 删除文章API
    └── categories.php           # 分类管理API
```

### 数据库表结构

**cms_articles（文章表）**
```sql
- id: 主键
- title: 标题
- summary: 摘要
- content: 内容
- category_id: 分类ID
- cover_image: 封面图
- status: 状态(draft/published/deleted)
- is_top: 是否置顶
- sort_order: 排序
- view_count: 浏览量
- seo_title: SEO标题
- seo_keywords: SEO关键词
- seo_description: SEO描述
- created_at: 创建时间
- updated_at: 更新时间
```

**cms_categories（分类表）**
```sql
- id: 主键
- name: 分类名称
- description: 描述
- sort_order: 排序
- created_at: 创建时间
```
