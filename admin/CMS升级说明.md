# CMS系统功能升级说明

## 升级内容概览

本次CMS系统功能升级包含以下5大核心功能：

### 1. 新增/删除页面功能 ✅
- **页面管理后台** (`admin/index.html`)
  - 显示所有页面列表（系统页面 + 自定义页面）
  - "新建页面"按钮，支持创建自定义页面
  - 删除页面功能（系统页面受保护，不可删除）
  - 页面统计信息展示

- **新建页面表单**
  - 页面ID（URL标识）
  - 页面名称
  - 页面标题（SEO）
  - 页面描述（SEO）
  - 页面模板选择

### 2. 元素级编辑功能 ✅
- **可视化编辑器** (`admin/editor-v2.html` + `admin/assets/editor-v2.js`)
  - 点击页面元素即可选中
  - 元素工具栏：编辑文本、更换图片、编辑链接、编辑样式、删除
  - 实时预览，所见即所得
  - 支持编辑的元素类型：
    - 文本元素（h1-h6, p, span, li等）
    - 图片元素（img）
    - 链接元素（a）
    - 按钮元素（button）

### 3. 图片上传/替换功能 ✅
- **图片库管理**
  - 本地上传图片（支持JPG, PNG, GIF, WebP）
  - 最大5MB限制
  - 图片预览功能
  - 已上传图片列表展示
  
- **图片使用场景**
  - 表单字段中的图片选择
  - 元素编辑时的图片替换
  - 图片URL自动填充

### 4. 修复核心业务领域正文编辑问题 ✅
- **富文本编辑器修复** (TinyMCE)
  - 正确绑定编辑器到文本域
  - 支持HTML内容编辑
  - 列表、链接、图片等富文本功能正常
  - 数据正确收集和保存

### 5. 修复富文本编辑器样式功能 ✅
- **完整的工具栏功能**
  - 加粗 (Bold)
  - 斜体 (Italic)
  - 下划线 (Underline)
  - 删除线 (Strikethrough)
  - 文字颜色 (ForeColor)
  - 背景颜色 (BackColor)
  - 左对齐/居中/右对齐/两端对齐
  - 无序列表/有序列表
  - 插入链接
  - 插入图片
  - 查看/编辑源代码

## 使用说明

### 访问页面管理
1. 登录CMS后台
2. 点击"页面管理"菜单
3. 查看所有页面列表
4. 点击"新建页面"创建新页面

### 编辑页面内容
1. 在页面列表中点击"编辑"
2. 使用左侧表单编辑内容
3. 富文本编辑器支持完整的样式功能
4. 点击"图片库"上传和管理图片
5. 右侧实时预览编辑效果
6. 点击"保存"或"发布"

### 元素级编辑
1. 在预览区域点击任意元素
2. 元素会被高亮选中
3. 显示元素工具栏
4. 选择编辑操作（文本/图片/链接/样式/删除）
5. 在弹出的模态框中编辑
6. 保存后实时生效

### 图片管理
1. 点击"图片库"按钮
2. 点击上传区域选择本地图片
3. 或从已上传图片中选择
4. 图片自动保存到图片库

## 技术特性

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **编辑器**: TinyMCE 6
- **存储**: LocalStorage (客户端) + JSON文件
- **图片**: Base64编码存储（本地模式）
- **响应式**: 支持桌面和移动端

## 注意事项

1. 系统页面（首页、业务范围等）受保护，不可删除
2. 图片上传在本地模式下使用Base64编码，建议定期导出备份
3. 所有更改默认保存到浏览器本地存储
4. 建议定期使用"导出数据"功能备份内容
