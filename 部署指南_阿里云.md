## 🔧 部署步骤

### 步骤1：准备服务器环境

确保阿里云服务器已安装以下环境：

```bash
# 1. 检查PHP版本（需要PHP 7.4+）
php -v

# 2. 检查MySQL版本（需要MySQL 5.7+）
mysql -V

# 3. 检查Nginx/Apache
nginx -v  # 或 httpd -v

# 4. 检查Node.js（用于图片上传服务）
node -v
npm -v
```

### 步骤2：创建网站目录

```bash
# 登录服务器后执行
sudo mkdir -p /www/wwwroot/api.yaozijin.com
sudo chown -R www:www /www/wwwroot/api.yaozijin.com
```

### 步骤3：上传文件到服务器

#### 方式A：使用SCP命令上传

```bash
# 在本地执行，上传admin目录
scp -r admin/ root@47.95.236.85:/www/wwwroot/api.yaozijin.com/

# 上传cms目录
scp -r cms/ root@47.95.236.85:/www/wwwroot/api.yaozijin.com/

# 上传server.js
scp server.js root@47.95.236.85:/www/wwwroot/api.yaozijin.com/
```

#### 方式B：使用FTP/SFTP工具

使用FileZilla、WinSCP等工具连接服务器，将文件上传到 `/www/wwwroot/api.yaozijin.com/` 目录。

### 步骤4：设置文件权限

```bash
# 登录服务器后执行

# 进入网站目录
cd /www/wwwroot/api.yaozijin.com

# 设置目录权限
sudo chown -R www:www /www/wwwroot/api.yaozijin.com
sudo chmod -R 755 /www/wwwroot/api.yaozijin.com

# 设置上传目录可写权限
sudo chmod -R 777 /www/wwwroot/api.yaozijin.com/cms/uploads
sudo chmod -R 777 /www/wwwroot/api.yaozijin.com/admin/data

# 设置PHP文件权限
sudo chmod -R 644 /www/wwwroot/api.yaozijin.com/admin/api/*.php
sudo chmod -R 644 /www/wwwroot/api.yaozijin.com/admin/api/**/*.php
```

### 步骤5：配置Nginx

创建或修改Nginx配置文件：

```bash
sudo nano /www/server/panel/vhost/nginx/api.yaozijin.com.conf
```

添加以下配置：

```nginx
server {
    listen 80;
    server_name api.yaozijin.com 47.95.236.85;
    root /www/wwwroot/api.yaozijin.com;
    index index.html index.php;

    # 前端页面
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 后台管理
    location /admin {
        try_files $uri $uri/ =404;
    }

    # PHP API接口
    location ~ \.php$ {
        fastcgi_pass 127.0.0.1:9000;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    # 上传文件访问
    location /cms/uploads {
        alias /www/wwwroot/api.yaozijin.com/cms/uploads;
    }

    # 禁止访问敏感文件
    location ~ /\. {
        deny all;
    }
    location ~ /(config\.php|init-db\.php) {
        deny all;
    }
}
```

重启Nginx：

```bash
sudo nginx -t
sudo systemctl restart nginx
# 或
sudo /etc/init.d/nginx restart
```

### 步骤6：初始化数据库

```bash
# 登录服务器后执行

# 进入网站目录
cd /www/wwwroot/api.yaozijin.com

# 运行数据库初始化脚本
php admin/api/init-db.php
```

如果命令行执行失败，可以通过浏览器访问：
```
http://47.95.236.85/admin/api/init-db.php
```

### 步骤7：启动Node.js图片上传服务

```bash
# 登录服务器后执行

# 进入网站目录
cd /www/wwwroot/api.yaozijin.com

# 安装依赖（首次需要）
npm install express multer cors

# 使用PM2启动服务（推荐，后台运行）
npm install -g pm2
pm2 start server.js --name "cms-upload-server"
pm2 save
npm2 startup

# 或使用nohup启动
nohup node server.js > server.log 2>&1 &

# 查看服务状态
pm2 status
# 或
ps aux | grep node
```

## ❗ 常见问题

### 1. 数据库连接失败

**问题**：无法连接到MySQL数据库

**解决**：
```bash
# 检查MySQL是否运行
sudo systemctl status mysql

# 检查用户权限
mysql -u root -p
GRANT ALL PRIVILEGES ON yaozijin.* TO 'yaozijin'@'localhost' IDENTIFIED BY 'fdsajkl';
FLUSH PRIVILEGES;
```

### 2. 文件上传失败

**问题**：无法上传图片

**解决**：
```bash
# 检查目录权限
sudo chmod -R 777 /www/wwwroot/api.yaozijin.com/cms/uploads

# 检查PHP上传限制
# 修改 php.ini
upload_max_filesize = 10M
post_max_size = 10M
```

### 3. Node.js服务无法启动

**问题**：端口被占用

**解决**：
```bash
# 查看端口占用
sudo netstat -tlnp | grep 3000

# 结束占用进程
sudo kill -9 <PID>

# 或修改server.js使用其他端口
```

### 4. 403 Forbidden错误

**问题**：访问被拒绝

**解决**：
```bash
# 检查文件权限
sudo chown -R www:www /www/wwwroot/api.yaozijin.com
sudo chmod -R 755 /www/wwwroot/api.yaozijin.com
```
